Articles: A / An / The

Артикль Коли? Приклад
A Перед приголосними a bug, a dev
An Перед голосними an apple, an API
The Щось конкретне the code is fixed
💡 A / An — тільки для однини!
💡 The — для всього, якщо ми знаємо, про що саме мова.

Articles: 7 Правил

Правило Коли? Приклад
a) A / An Злічуваний іменник в однині I've got a car
b) --- Іменники у множині I don't like buses
c) --- Міста та країни Madrid is in Spain
d) --- Усталені фрази без артикля by bus, on foot, at work
e) The Єдиний у своєму роді the moon, the Eiffel Tower
f) The Усталені фрази з the in the morning, on the right
g) The Найвищий ступінь порівняння the most beautiful, the worst
💡 Без артикля = поясни чому!
💡 --- у квізі = без артикля
